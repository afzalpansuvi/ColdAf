# How to Create an Email Template in ColdAF

**Last Update:** July 2026

**Post ID:** 144

A strong email template is the foundation of effective cold outreach. Instead of writing every message from scratch, you can **create email template coldaf** once and reuse it across campaigns. This guide walks through the **cold email template builder** so you can craft polished, personalized emails at scale. Learning how to master **outreach email template creation** will save you time and ensure consistency across your team while still allowing room for personalization through variables and spintax.

## You can create an email template in just a few steps. Here's how to do it:

1. Go to **Templates** from the main navigation.
2. Click **Create Template** to start a new template.
3. Enter a template name that describes the purpose, for example, **Intro to SaaS Founders**.
4. Select a category from the dropdown, such as **Introduction**, **Follow-up**, or **Value Add**.
5. Write the subject line for the template.
6. Write the email body in the editor.
7. Use variables like **{{firstName}}**, **{{company}}**, and **{{role}}** to personalize the message for each lead.
8. Add spintax for variation if desired, using syntax like **{{quick question|quick idea}}**.
9. Click **Save Template** to store it in your library.
10. Preview the template with sample data to see how it will look when sent to a real lead.

That's it! Now you can **create email template coldaf** and build a reusable library of high-quality outreach messages.

## Notes

- Templates are reusable across multiple campaigns, so invest time in writing strong ones.
- Categories help you organize templates by use case and make them easier to find later.
- Variables auto-populate with lead data at send time, so every email feels personal.
- Spintax syntax is **{{option1|option2}}**. Each send randomly picks one option for natural variation.
- AI can generate the full template from a simple prompt if you want a fast starting point.
- The preview panel shows how the template looks with real lead data, so you can check formatting before sending.
- Templates can be edited anytime, and updates apply to future campaigns that use them.
