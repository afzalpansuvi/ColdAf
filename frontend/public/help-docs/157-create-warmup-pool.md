# How to Create a Warmup Pool in ColdAF

**Last Update:** July 2026

**Post ID:** 157

When you are managing multiple brands or client accounts, warming them up individually becomes tedious and inefficient. A **warmup pool** lets you group related accounts together so they share a collective reputation strategy and ramp schedule. This is especially useful for agencies and teams running **cold outreach** at scale. By creating a **warmup account pool** in ColdAF, you can apply shared settings, monitor progress across the group, and reduce the manual overhead of managing each account separately. This article explains how to set up a pool and assign accounts to it.

## You can create a warmup pool in just a few steps. Here's how to do it:

1. Go to the **Warmup** page from the main navigation.
2. Click the **Pools** tab to access pool management.
3. Click **Create Pool** to start a new pool.
4. Enter a name for the pool so you can identify it later.
5. Select the accounts you want to add. Pick from your existing **SMTP Accounts** list.
6. Set the pool-level strategy that will apply to all members of the group.
7. Set the shared target volume for the entire pool.
8. Click **Create Pool** to save the pool.
9. Assign warmup campaigns to this pool when you create or edit them.

That's it! Now you can manage grouped warmup campaigns and keep your accounts organized.

## Notes

- **Pools** group accounts that warm together under a single strategy and schedule.
- This feature is great for agencies managing multiple client accounts in one ColdAF organization.
- Accounts within a pool share reputation signals, which can strengthen collective deliverability.
- You can move accounts between pools at any time without stopping their warmup.
- Each pool has its own stats and progress dashboard so you can track performance at a glance.
- You can remove an account from a pool without stopping its individual warmup campaign.
- Pool-level settings override individual account settings, so make sure your pool configuration matches your goals.
